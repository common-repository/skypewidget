Allows to display the skype buttons and online state on the sidebar as a widget.

== Description ==
<p>
In the Themes->Sidebar Widget you see in the available widgets section the SkypeWidget, which you can drag and drop on your side bar.<br> 
You can add an Title which should be displayed in the title of the widget.<br>
Add the user account which should be used to retrieve the online state and functionality.<br>
Select which functionality should be visible on the widget.<br>
This widget creates the button html code which is also created from the <a href="http://www.skype.com/share/buttons/wizard.html">skype button page</a> 
and uses the buttons created by the skype pages.<br>
To activate the online state you need to configure your <a href="http://www.skype.com/share/buttons/status.html">skype account</a> to allow this.
</p> 

== Installation ==

1. Unzip the file  you downloaded
2. Copy the SkypeWidget.php file in your Sidebar Widget folder
3. Enable the plugin in your Plugin administration page.
4. Drag the SkypeWidget on your Sidebar
5. Configure your SkypeWidget (See "Advaned Usage" section)


== Configuration ==
* Set the title
* Set the user
* Select the buttons you like to provide.
 * Show Call: Show the call me button.
 * Show Add:  Show the add me to skype button.
 * Show Chat: Show the chat with me button.
 * Show Info: how the view my profile button.
 * Show Voicemail: Show the leave me a voicemail button.
                   You need an additional Skype subscription for receiving voidemails see <a href="http://www.skype.com/products/skypevoicemail/">Voicemail subscription</a>
 * Show Sendfile: Show the send me a file button.
* Select the colour schema (green or blue)
