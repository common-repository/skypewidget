SkypeWidget

The SkypeWidget Plugin is easy to integrate and maintainable provider for your Skype online status and functionality.

Installation:
Unzip the file you downloaded and copy the SkypeWidget.php file in your Sidebar Widget folder.
Enable the plugin in your Plugin administration page.

Usage:

In the Themes->Sidebar Widget you see in the available widgets section the SkypeWidget, which you can drag and drop on your side bar.
You can add an Title which should be displayed in the title of the widget.
Add the user account which should be used to retrieve the online state and functionality.
Select which functionality should be visible on the widget.
This widget creates the button html code which is also created from the skype button page and uses the buttons created by the skype pages.
To activate the online state you need to configure your skype account to allow this.

I use the small icon set because this set only has transparent backgrounds set. The big icon set has a white background and looks ugly in the most thems I tried.

FunctionDescription
Show CallShow the call me button.
Show AddShow the add me to skype button.
Show ChatShow the chat with me button.
Show InfoShow the view my profile button.
Show VoicemailShow the leave me a voicemail button.
You need an additional Skype subscription for receiving voidemails see Voicemail subscription
Show SendfileShow the send me a file button.

You can select the colour schema which should be used the green or blue one.
